# 主控板使用文档
 
![主控板](img/spdrw.png)


## RST 和 BOOT 按键
- **RST**：主控复位按键，按下松开主控复位，恢复初始状态。
- **BOOT**：功能键，连按两次启动 AP 模式，`WIFI` 名称 `SPDRW-WS`，密码 `123456789`。
- 启动 AP 模式时同步启动 WS 服务端，可通过 WS 客户端（例如安卓端）连接 `ws://192.168.5.1`。

## RGB 指示灯
- 根据操作结果显示不同颜色，例如 `write` 操作写入失败显示红色，写入成功显示绿色。

## AP 指示灯
- AP 模式启动时常亮。
