# SPDRW

 - 根据开源项目 https://github.com/1a2m3/SPD-Reader-Writer 做的配套硬件；利用Arduino对I2C总线进行读写，一块主控板搭配不同的DIMM支持所有的DDR内存。

# 功能

 - 支持所有DDR内存；
 - 支持解除写保护；
 - 支持导出导入SPD bin文件；

## 标准现货 

 - 2025-2-21更新
 - 发货时间T+1

|  类型   | 价格(元)  | 单位 | 库存 |
|  ----  | ----  | ---- | ---- |
| 主控板  | 349 | 1PCS | 13 |
| DIMM-D3-D4  | 50 | 1PCS | 1 |

## 硬件定制开发

 - DDR4 支持 8DIMM 读写，带写保护仅支持 4DIMM.
 - DDR5 支持 8DIMM 读写及写保护。
 
## 软件开发
 - 支持各字节指定修改。
 - 扫描枪输入修改指定字节
 - 批量修改。
